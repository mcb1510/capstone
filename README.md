**Author:** Miguel Carrasco  
**Institution:** Boise State University  
**Program:** M.S. in Computer Science

This repository contains the public preprint version of my capstone final report and related project materials.

[capstone_final_report.pdf](./capstone_final_report.pdf)

## Project Summary
This project presents a retrieval-augmented advisor recommendation assistant for Boise State University Computer Science graduate students. The system retrieves grounded faculty evidence and uses structured generation to support advisor discovery.
